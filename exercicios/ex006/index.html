Tipos de Marcações

Negrito

Para deixar uma área em negrito , como o parte anterior nescessita de usar a tag <strong >.


Podemos usar também a tag <b>, mas essa não é recomendada já que é uma tag apenas visual, sem significado, é melhor deixar o visual para a linguagem CSS.

Itálico

Para usarmos itálico, se usa a ta <em>


Antigamente era usado a tag <i>, mas essa tag é apenas visual e para visual temos o CSS, no html, queremos significado no hipertexto.

Marca Texto

Para usar um marca texto, usamos a tag <mark>.

small

A tag <small>,deixa o texto pequeno, ja a <big>, não é usada mais.

Texto "acima" e "baixo"

H2O, usamos <sub>

X2, usamos <sup>

Texto Deletado

Para "cortamos um texto", desse jeito, usamos a tag <del>

Sublinhado

Para sublinhar, como fizemos, usa-se a tag <ins>
